Catalog Item: Mobile Request

Activity 1:
Manager Approval
- Ask for approval from Requested For's Manager.
- If approved, create three dependent tasks:
  1. SIM Card Allocation assigned to Telecom Team.
  2. Mobile Device Configuration assigned to IT Support.
  3. Device Delivery assigned to Logistics Team.
- All three tasks must execute in parallel after approval.
- If rejected, end the workflow.

Activity 2:
Device Setup Completion
- Create a task named Setup Completion Validation assigned to Operations Team as the individual starting action.
- After Setup Completion Validation is Closed Complete, create Verification Task assigned to QA Team.
- After Verification Task is Closed Complete, create Security Compliance Task assigned to Security Team.
- Security Compliance Task must depend on Verification Task.
- All dependencies must remain within this activity only.
- If any task is Closed Incomplete, end the workflow.

Activity 3:
Final Closure
- Create a task named Final Process Review assigned to Service Desk as the individual starting action.
- After Final Process Review is Closed Complete, create Final Closure Task assigned to Service Desk.
- If Final Closure Task is Closed Incomplete, end the workflow.
